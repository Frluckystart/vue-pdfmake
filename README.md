# vue-pdfmake

基于 pdfmake
![](https://travis-ci.com/mizuka-wu/vue-pdfmake.svg?branch=master)

## 一个 vue 组件，能够将内部的 slot 转为 pdf

项目基于 VNODE 将 vnode 按照一定的规则转为 pdfmake 的相关配置文件

![预览](docs/public/preview.png)

## 文档

[查看文档](https://www.mizuka.top/vue-pdfmake/)

```
yarn install
```
